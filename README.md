# Install

Clone this project:

```
git clone https://github.com/MarianoGarciaGz/MarianoGarciaGz.github.io.git
```

Access:

```
cd ./MarianoGarciaGz.github.io
```

Install dependencies:

```
npm i
```

I use this node modules: [gh-pages](https://www.npmjs.com/package/gh-pages), sass, [swipper](https://www.npmjs.com/package/swiper), aos animations

Run:

```
npm run dev
```

Deploy:

```
npm run build
```

&

```
npm run deploy
```
